Options

/list - List all if passed without state
/list/SP - List all students filter by state = SP (or others)
/create - Create a student by giving a JSON body {"name": "some_name", "cpf": 123, "state": "SP"}
/delete - Delete a student with cpf passing a body in format {"cpf": valid_number}
/update - Update all data from student with cpf field {"name": "new_name", "cpf": 123, "state": "SC"}
